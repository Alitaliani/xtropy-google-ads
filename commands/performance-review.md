# Google Ads Performance Review

Quick performance check for a Google Ads account.

## Target Account

$ARGUMENTS — provide a customer ID (e.g., 396-860-2949) or account name.

## Workflow

1. Use `list_accounts` to verify the account
2. Use `get_account_currency` to understand cost units
3. Run `get_campaign_performance` for the last 30 days
4. Run `get_budget_pacing` to check spend efficiency
5. Run `get_competitive_position` for impression share trends

Summarize: total spend, conversions, CPA, top/bottom performing campaigns, and any immediate concerns.
